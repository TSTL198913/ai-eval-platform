## 项目概况

### 项目背景

**AI Eval Platform** 是一个分布式AI评测系统,用于评估AI模型的质量、安全性和性能。项目包含:

- **前端**: React + TypeScript + Ant Design
- **后端**: Python + FastAPI + SQLAlchemy
- **测试**: pytest + vitest + playwright
- **部署**: Docker + Kubernetes + Prometheus

### 项目架构

```
AI Eval Platform
├── 前端层 (React)
│   ├── Dashboard (仪表盘)
│   ├── Evaluators (评估器管理)
│   ├── Records (评估记录)
│   ├── Models (模型管理)
```
