## 1. Product Overview

AI全链路评测系统是一个分布式AI模型评估平台，提供多维度评估器、实时监控、成本治理和模型对比等核心能力。本次迭代目标是实现前后端完全分离，构建现代化的Web前端界面，提升用户体验和系统可维护性。

**目标用户**: AI模型开发者、QA测试工程师、运维人员、产品经理

**核心价值**: 提供直观的可视化界面，让用户能够轻松管理评估任务、查看评估结果、对比不同模型性能、监控系统健康状态和成本支出。

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |

6. **成本监控**: 费用统计、预算管理、成本趋势

## 3. Core Process

### 用户登录流程
1. 用户访问登录页面
2. 输入用户名和密码
3. 系统验证身份并返回JWT Token
4. Token存储在localStorage，后续请求携带Authorization头
5. 跳转到仪表盘首页

### 评估任务流程
1. 用户在仪表盘查看统计概览
2. 选择评估器并提交评估任务
3. 系统异步处理任务
4. 用户查看任务状态和结果
5. 在评估记录中查看历史数据

## 4. User Interface Design

### 4.1 Design Style
- **主色调**: 深邃科技蓝 (#1e3a5f)，搭配渐变紫色 (#667eea  #764ba2)
- **辅助色**: 绿色成功态 (#10b981)，红色错误态 (#ef4444)，橙色警告态 (#f59e0b)
- **按钮风格**: 圆角矩形 (border-radius: 8px)，hover状态有阴影加深
- **字体**: Inter字体系列，标题使用600-700权重，正文使用400权重
- **布局**: 卡片式布局，左侧固定导航栏，右侧主内容区
- **图标**: Lucide React图标库，统一24px大小

### 4.2 Page Design Overview

| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 登录页 | 登录表单 | 居中卡片布局，渐变背景，表单验证提示，加载状态 |
| 仪表盘 | 统计卡片 | 6个数据卡片，网格布局，hover动画效果，数值动画 |
| 仪表盘 | 状态分布 | 横向柱状图，颜色编码，数值标签 |
| 仪表盘 | 快速入口 | 标签式评估器列表，点击跳转 |
| 评估器管理 | 评估器列表 | 卡片网格，悬停展示详情，搜索框 |
| 模型对比 | 对比图表 | 柱状图，模型颜色区分，图例 |
| 评估记录 | 记录列表 | 表格形式，状态徽章，操作按钮 |
| 成本监控 | 成本概览 | 数字卡片，进度条展示预算使用 |
| 系统健康 | 组件状态 | 网格卡片，状态指示灯，详细信息 |

### 4.3 Responsiveness
- **桌面端**: 完整功能布局，左侧导航栏 + 右侧主内容区
- **平板端**: 响应式布局，导航栏折叠为汉堡菜单
- **移动端**: 单列布局，卡片堆叠，触控优化按钮尺寸

### 4.4 Accessibility
- 所有交互元素有明确的hover和focus状态
- 颜色对比度符合WCAG AA标准
- 表单元素有label关联
- 键盘导航支持
