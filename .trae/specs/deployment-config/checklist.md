# AI 评测平台 - 部署配置验证检查清单

## 配置文件检查

### 必需文件
- [ ] .env.prod 文件存在且包含所有必需变量
- [ ] docker-compose.prod.yml 文件存在且格式正确
- [ ] deploy-prod.sh 脚本存在且可执行
- [ ] SSL/TLS 证书文件存在（生产环境）

### 环境变量检查
- [ ] 数据库连接配置（POSTGRES_USER, POSTGRES_PASSWORD, DATABASE_URL）
- [ ] RabbitMQ 连接配置（RABBITMQ_USER, RABBITMQ_PASSWORD, RABBITMQ_URL）
- [ ] Redis 连接配置（REDIS_URL）
- [ ] LLM API Key 配置（DEEPSEEK_API_KEY 或 OPENAI_API_KEY）
- [ ] 环境标识配置（ENV=production）

## 安全检查

### 敏感信息
- [ ] 数据库密码不为默认值
- [ ] RabbitMQ 密码不为默认值
- [ ] Redis 已配置密码认证
- [ ] LLM API Key 不为空且通过环境变量注入
- [ ] 敏感信息未硬编码在代码或配置文件中

### 网络安全
- [ ] 生产环境启用 HTTPS
- [ ] CORS 策略配置合理
- [ ] 不必要的端口未暴露
- [ ] 防火墙规则配置正确

## 服务配置检查

### 数据库服务
- [ ] PostgreSQL 版本 >= 14
- [ ] 连接池大小配置合理（建议 10-20）
- [ ] 数据库持久化存储已配置
- [ ] 数据库备份策略已配置

### 消息队列服务
- [ ] RabbitMQ 版本 >= 3.10
- [ ] 虚拟主机已创建
- [ ] 队列持久化已启用
- [ ] 独立用户权限已配置

### 缓存服务
- [ ] Redis 版本 >= 7.0
- [ ] 密码认证已启用
- [ ] 缓存过期策略已配置

### 应用服务
- [ ] API 服务端口配置正确（建议 8000）
- [ ] 健康检查端点 /health 已配置
- [ ] 监控指标端点 /metrics 已配置
- [ ] 资源限制已配置（CPU/内存）
